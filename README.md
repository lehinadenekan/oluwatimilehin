# Timilehin Portfolio Website

A professional portfolio website built with Next.js 14, React, TypeScript, and Tailwind CSS.

## Features

- Dark theme with black/gray color scheme
- Purple and pink accent colors
- Collapsible sidebar navigation (Google Maps style)
- Typewriter animation in hero section
- Spotify embeds for music releases
- Fully responsive design
- Smooth scroll navigation
- Active section highlighting

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is ready to deploy to Vercel. Simply connect your GitHub repository to Vercel and it will automatically deploy.
